# seoul-commercial-district-analysis

## Objective

Data analysis project exploring factors affecting commercial sales growth in Seoul using Python, regression analysis, and GIS.

## Tools

- Python
- Pandas
- Statsmodels
- GeoPandas

## Methods

- Multiple Regression
- VIF
- t-test
- GIS Analysis

## Results

- Floating population positively affects sales.
- Rent negatively affects sales growth.
